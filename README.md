# Public Scripts

This repository is a collection of personal scripts, utilities, experiments, and workflow tools accumulated over time.

Most of these were written to solve specific problems in my own environment and may reflect assumptions about directory structures, naming conventions, installed software, operating systems, or workflow habits that exist on my machines and nowhere else.

Some scripts may:

* modify files
* move or rename data
* create or delete content
* make assumptions that are correct for my workflow but not yours
* be unfinished or quickly assembled for a one-time task

These scripts are provided as-is. If you decide to use them, read them first and understand what they do before running them against important data.

Some may be useful to others, some may simply be interesting artifacts, and some may serve mainly as reminders of how I solved a problem at a particular moment in time.

If something here helps you, great.
